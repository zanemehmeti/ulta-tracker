# Ulta Beauty Gratis & Purchase Tracker

A simple web application to track Ulta Beauty products received as **gratis** or **purchases** per user using browser localStorage.  

---

## Features

- User-based tracking (each username has its own data)
- Add, edit, and delete products
- Categorize products by **Makeup, Skincare, Haircare, Fragrance**
- Mark products as **Gratis** or **Purchase**
- Saves data locally in the browser (no backend required)

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- jQuery
- localStorage

---

## How to Use

1. Open `index.html` in your browser.
2. Enter a **username** and click **Enter Tracker**.
3. Add new items with product name, category, and type.
4. Edit or delete items as needed. Your data is saved locally in your browser.
